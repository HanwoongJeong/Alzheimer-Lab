1. pet_augmentation_xml.py : 자료 뻥튀기 기능
2. pet_image_process.py : 히스토그램 이용해서 등고선처럼 이미지 변형

1-1)origin_augmented.zip  : origin 폴더의 자료를 각각 10,000개씩 augmentation시켜 총 30,000개 파일 생성

1-2)simple.zip : origin 폴더의 자료를 컬러값을 단순화시켜서 등고선이 생기게 만들어 봤습니다. 1gr, 2gr, 3gr 을 구분하는데 도움이 될지 테스트가 필요합니다.
실험은 제가 해보겠지만 이 이미지가 의사가 분별하는데 유의미한지 교수님이 한 번 봐주셨으면 합니다.

1-3)origin_augmented.zip  : origin 폴더의 자료를 각각 노이즈, 조명효과 빼고 50,000개씩 augmentation시켜 총 15,000개 파일 생성

1-4)simple_augmented.zip  : simple 폴더의 자료를 각각 노이즈, 조명효과 빼고 50,000개씩 augmentation시켜 총 15,000개 파일 생성
